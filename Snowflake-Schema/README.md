## Snowflake Schema Data Model

This Power BI report demonstrates a Snowflake Schema data model.

### Data Source
- Snowflake_Schema_Tables.xlsx

### Description
The Excel file contains normalized tables such as fact and multiple
related dimension tables. These tables were loaded into Power BI
and connected using relationships to form a snowflake schema.

### Key Features
- Normalized dimension tables
- Multiple relationships from fact to dimensions
- Reduced data redundancy
- Structured and scalable data model

### Purpose
To demonstrate understanding of snowflake schema design and
data modeling techniques in Power BI.
