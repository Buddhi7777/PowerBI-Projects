# PowerBI-Projects

This repository contains Power BI projects created to demonstrate
data visualization, dashboarding, and data modeling concepts.

## Projects Included

### 1. Charts and Dashboard
- Focuses on creating charts and interactive dashboards
- Built using Excel sales and orders data
- Demonstrates data cleaning, visual design, and KPI analysis

### 2. Snowflake Schema Data Model
- Demonstrates a snowflake schema data model in Power BI
- Uses normalized dimension tables
- Focuses on relationship management and structured data modeling

### 3. Star Schema (Flat Table) Data Model
- Demonstrates a star schema with a flat fact table
- Optimized for reporting and performance
- Shows differences between star and snowflake schemas

## Data Sources
- Excel datasets sourced from Kaggle (public datasets)
- Used strictly for learning and educational purposes

## Tools & Technologies
- Power BI Desktop
- Microsoft Excel
- Power Query
- Data Modeling (Star Schema & Snowflake Schema)

## Purpose
The purpose of this repository is to showcase practical Power BI skills,
including data visualization, dashboard creation, and analytical data modeling.
