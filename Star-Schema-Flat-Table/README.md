## Star Schema (Flat Table) Data Model

This Power BI report demonstrates a Star Schema using a flat fact table.

### Data Source
- Star_Schema_Tables.xlsx

### Description
The Excel data contains a central fact table along with denormalized
dimension data. Relationships were created in Power BI to form
a star schema model.

### Key Features
- Central fact table
- Denormalized dimensions
- Simple relationships
- Faster query and report performance

### Purpose
To demonstrate star schema modeling and performance-oriented
data design using Excel data in Power BI.
