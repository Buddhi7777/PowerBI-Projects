## Charts and Dashboard

This Power BI report focuses on data visualization and dashboard creation.

### Data Sources
- SalesData.xlsx
- Orders_data.xlsx

### Description
The Excel data was imported into Power BI and transformed using Power Query.
Multiple charts were created to analyze sales and order performance.

### Key Visuals
- Sales trends over time
- Order count analysis
- Category-wise and region-wise comparisons
- KPI indicators for overall performance

### Purpose
To demonstrate Power BI chart creation, dashboard layout design,
and basic data transformation using Excel data sources.
